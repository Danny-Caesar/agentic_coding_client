# Setup Manager Agent

## Purpose
This agent manages the complete development environment setup for the Agentic Coding Client project, including:
- System dependencies installation
- Development tools configuration
- VS Code setup and extensions
- Build environment preparation
- Package management (vcpkg)

## Capabilities

### 1. Environment Detection
- Detects OS (Ubuntu/WSL2)
- Checks for existing tools and versions
- Validates system requirements

### 2. Installation Management
- Installs system packages via apt
- Configures vcpkg for C++ dependency management
- Sets up development tools (CMake, g++, gdb)
- Installs Qt and OpenCV libraries

### 3. VS Code Configuration
- Installs recommended extensions
- Configures workspace settings
- Sets up debugging profiles
- Configures C++ intellisense

### 4. Project Setup
- Generates initial project structure
- Creates CMakeLists.txt templates
- Sets up vcpkg configuration
- Initializes Git repository

## Workflow

1. **Preparation**: Analyze project requirements and current system state
2. **Installation**: Execute installation scripts and commands
3. **Configuration**: Set up development tools and IDE
4. **Validation**: Verify all installations and configurations
5. **Documentation**: Generate setup completion report

## Integration

This agent works with:
- `setup-dev-environment.skill.md` - Detailed installation instructions
- `copilot-instructions.md` - Project-wide coding guidelines
- `REQUIREMENTS.md` - Project specifications

## Usage

Invoke this agent when:
- Starting a new development environment
- Onboarding new developers
- Troubleshooting missing dependencies
- Upgrading development tools
