# scrum.skill.md

이 스킬은 스크럼 마스터 에이전트가 제품 백로그 관리, 스프린트 계획, 스토리 작성 및 우선순위 결정을 자동화하거나 보조할 때 사용하는 규칙과 템플릿을 제공합니다.

목차:
- 백로그 분석 템플릿
- 스프린트 계획 템플릿
- 수용 기준 템플릿
- 우선순위 및 추정 규칙

## 백로그 아이템 템플릿

```
- ID: PB-###
- 타이틀: 간결한 사용자 스토리
- 설명: 누가, 무엇을, 왜
- 우선순위: 높음/보통/낮음
- 추정(스토리 포인트): 1/2/3/5/8/13
- 수용 기준:
  - ...
- 노트:
  - 관련 에픽: EPIC-#
  - 담당자: TBD
```

## 스프린트 계획 템플릿

```
스프린트: Sprint #
기간: 2주
목표: 간단한 스프린트 목표 설명
커밋된 스토리:
 - PB-### : 타이틀 (SP)
 - PB-### : 타이틀 (SP)
용량 추정: X SP
```

## 우선순위 규칙
- Must(높음): 제품 기본 동작에 필수
- Should(보통): 중요한 개선이나 안정성 기능
- Could(낮음): 부가적 개선
